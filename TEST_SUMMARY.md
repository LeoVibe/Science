## 📝 注意事项

1. **题目数量**：
   - 目前除了自然科（100题），其他科目都只有5题示例
   - 如果需要更多题目，可以在对应的 `questions_*.js` 文件中添加

2. **数据存储**：
   - 每个科目的数据独立存储
   - 切换科目时，数据不会混淆

3. **Favicon**：
   - 使用 SVG 格式，现代浏览器支持
   - 如果浏览器不支持 SVG，会回退到 ICO 格式

## 🔧 如果遇到问题

1. **构建失败**：
   - 检查 Node.js 版本（需要 18+）
   - 运行 `npm install` 重新安装依赖
   - 检查是否有语法错误

2. **题目不显示**：
   - 检查浏览器控制台是否有错误
   - 确认题目文件格式正确
   - 检查科目是否正确选择

3. **数据不保存**：
   - 检查浏览器是否允许 localStorage
   - 清除浏览器缓存后重试
